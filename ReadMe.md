### RestAPISample



This is a VS2022 ASP.NET Core MVC REST API. It serves as the API to a Microsoft SQL Server on Azure.

This REST API is called to by the School Web App ([https://schoolwebapp3rda2026.azurewebsites.net/](https://schoolwebapp3rda2026.azurewebsites.net/)) to display data on the Staff and Courses pages.
